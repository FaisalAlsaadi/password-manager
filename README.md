A Python Tkinter-module based GUI password generator/manager. Encrypted with AES-256 using the profile password. 
Have I been Pawned intergration: if API key is provided in the settings it can query the HIBP database for all user emails/passwords to check if they in any breaches.
Various theme options are included.
Password strength indicator.
Completely local. Profiles are created/stored in the same directory as the application.
